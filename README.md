# 🎂 Age Calculator Web App

A simple and modern **Age Calculator** built using **HTML, CSS, and JavaScript**.
This web app calculates a user's exact age in **years, months, and days** based on their date of birth.

---

## ✨ Features

* 📅 Select your **date of birth**
* 🧮 Calculates **exact age**
* 📊 Shows **years, months, and days**
* 🎨 Clean **modern UI**
* 📱 Fully **responsive design**
* ⚡ Instant calculation using **JavaScript**

## 🧠 How It Works

1. User selects their **date of birth**.
2. JavaScript calculates the difference between the current date and birth date.
3. The app displays the **exact age in years, months, and days**.
